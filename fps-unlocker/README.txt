This mod requires the Jeode mod loader (https://github.com/Paficent/Jeode)
Your mods folder should should like `mods/fps-unlocker/manifest.json` etc.

The settings.json is where you can set a higher FPS value.

The source code for this mod is available at https://github.com/TheModderHandlers/fps-unlocker/tree/main
